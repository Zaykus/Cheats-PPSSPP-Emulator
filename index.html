⚙️ Installation Guide

Step-by-step instructions for setting up cheats in PPSSPP

Step 1: Enable Cheats in PPSSPP

Launch PPSSPP

Open your PPSSPP emulator application on your device.

Access Settings

Navigate to Settings → System from the main menu.

Enable Cheat Support

In the System settings, locate the Cheats section and enable:

  • ☑️ Enable cheats
  • ☑️ Enable plugins (recommended for advanced features)
Important: Both options must be enabled for cheats to function properly in PPSSPP.

Step 2: Locate Your Cheats Directory

The cheats folder location varies by platform. Find your platform below:

Platform Default Cheats Directory
Windows Documents/PPSSPP/PSP/Cheats/
Android /storage/emulated/0/PSP/Cheats/
macOS ~/Library/Application Support/PPSSPP/PSP/Cheats/
Linux ~/.config/ppsspp/PSP/Cheats/
iOS (Jailbroken) /var/mobile/Documents/PPSSPP/PSP/Cheats/

Can't find the cheats folder?

  1. Open PPSSPP and go to Settings → System
  2. Look for Memory Stick folder or tap Show Memory Stick folder
  3. Navigate to [Memory Stick Path]/PSP/Cheats/
  4. If the Cheats folder doesn't exist, create it manually

🎮 Usage Instructions

How to download, install, and activate cheats

Method 1: Database Import (Recommended)

Download cheat.db

Get the latest database from trusted sources:

  • Official Repository
  • CWCheat Database Plus+
Install Database

Place the cheat.db file in your PPSSPP cheats directory:

PPSSPP/ └── PSP/ └── Cheats/ └── cheat.db
Import In-Game
  • Launch your game in PPSSPP
  • Open pause menu (ESC key or back button)
  • Select Cheats
  • Tap Import from cheat.db
  • Enable desired cheats and resume

Method 2: Manual Entry

Create Cheat File

Launch your game and access the cheats menu to auto-generate the .ini file named after your Game ID.

Edit INI File

Open the generated file in a text editor and add cheats in CWCheat format:

_S ULUS-10202 _G God of War - Chains of Olympus [USA] _C0 Infinite Health _L 0x200xxxxx 0x12345678 _C0 Max Orbs _L 0x200xxxxx 0x0098967F
Format Reference
  • _S = Game ID identifier
  • _G = Game title (optional)
  • _C0 = Cheat description
  • _L = Cheat code (address + value)

Activating Cheats

  1. Pause your game (ESC key or pause button)
  2. Select Cheats from the pause menu
  3. Use checkboxes to toggle cheats on/off
  4. Resume gameplay to see effects
Note: PPSSPP doesn't use visual status indicators for cheats. Active cheats are simply checked in the menu, inactive ones are unchecked. If a cheat isn't working, verify the Game ID matches your ROM region.

🔧 Troubleshooting

Common issues and solutions

Common Issues

Cheats not showing up
  • Verify the Game ID matches between your ROM and the cheat file
  • Ensure the cheat file is in the correct directory
  • Restart PPSSPP after adding new cheat files
Cheats not working
  • Make sure "Enable cheats" is checked in PPSSPP settings
  • Try disabling other cheats that might conflict
  • Some cheats require specific conditions to activate

Advanced Solutions

Game freezes when cheats are enabled
  • Update to the latest PPSSPP version
  • Try disabling all cheats except the one you need
  • Some cheats may be incompatible with your ROM version
Creating custom cheats
  • Use CWCheat Database Editor for cheat creation
  • Test cheats in small segments before combining
  • Always backup your save files before testing new cheats
